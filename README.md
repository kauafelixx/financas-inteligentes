# Finanças Inteligentes

Landing page institucional fictícia para uma consultoria financeira. O projeto apresenta serviços, processo de atendimento, depoimentos, artigos educativos e um formulário simples de captação de leads.

## Sobre o projeto

O site foi criado para treinar a construção de uma página empresarial com visual profissional, navegação por seções e foco em conversão. A proposta da marca é comunicar clareza, segurança e organização financeira para pessoas que querem planejar melhor renda, metas e investimentos.

## Funcionalidades

- Cabeçalho fixo com navegação interna.
- Hero section com chamada principal, botões de ação e indicadores de confiança.
- Seção de serviços para consultoria financeira, planejamento e investimentos.
- Bloco explicando o processo de atendimento em três etapas.
- Depoimentos fictícios com imagens de clientes.
- Cards de artigos sobre educação financeira.
- Formulário de diagnóstico gratuito com validação de e-mail.
- Animações suaves ao rolar a página.
- Layout responsivo para celular, tablet e desktop.
- Suporte a redução de movimento via `prefers-reduced-motion`.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Como executar

Como este é um site estático, não é necessário instalar dependências.

1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` no navegador.

Também é possível executar com a extensão Live Server do VS Code para ter recarregamento automático durante o desenvolvimento.

## Personalização

Você pode adaptar o projeto alterando:

- Textos, links e seções no arquivo `index.html`.
- Cores, espaçamentos, responsividade e animações no arquivo `home.css`.
- Comportamento das animações e mensagem do formulário no arquivo `anima.js`.
- Imagens, logotipo e ícones dentro da pasta `image/`.

## Observações

Os dados da empresa, endereço, depoimentos e resultados exibidos são fictícios e foram usados apenas para compor uma landing page de estudo.
